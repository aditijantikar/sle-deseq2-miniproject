# sle-deseq2-miniproject
This project involves finalizing the Systemic Lupus Erythematosus (SLE) mini-project, focusing on the Differential Gene Expression (DGE) analysis using the `DESeq2` package in R. The primary goal was to process RNA-seq data, identify differentially expressed genes, and prepare gene sets.
